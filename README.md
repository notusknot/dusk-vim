## Installation

- [dein](https://github.com/Shougo/dein.vim)
  ```viml
  call dein#add('yonlu/omni.vim')
  ```
  Install with: ``:call dein#install()``

  After installed, add this line to your vimrc.
  ```viml
  colorscheme omni
  ```
